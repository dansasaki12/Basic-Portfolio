About Me

Hello my name is Dan Sasaki, I currently work for Instructre as an Accessibility QA Engineer.
For this position, I specialize in ensuring that the code we release is accessible for users with disabilities. This can be a user with limited visibility so they'll utilize a screenreader to navigate our pages, a user that can only use the keyboard to navigate or any other users that utilize assitive technology to navigate webpages.
